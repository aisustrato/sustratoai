## 🎯 Problema Central

> *"Un listado lineal no refleja la necesidad de tejer significado."*

La acumulación lineal de artefactos replica el paradigma educativo tradicional: datos → más datos → saturación. 
Cognética debe implementar el paradigma opuesto: **conexiones inesperadas según áreas de interés**.

## 🗂️ Modelo de Datos Extendido

### Nuevas Tablas Necesarias

```sql
-- Granjas de Conceptos
CREATE TABLE cog_farms (
    id UUID PRIMARY KEY,
    project_id UUID REFERENCES cog_projects(id),
    name TEXT NOT NULL,
    type TEXT CHECK (type IN ('artifacts', 'seeds', 'thinkers', 'theories', 'custom')),
    description TEXT,
    icon TEXT, -- emoji
    color TEXT, -- hex
    position JSONB, -- {x, y} para visualización
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Membresía de elementos en granjas
CREATE TABLE cog_farm_members (
    id UUID PRIMARY KEY,
    farm_id UUID REFERENCES cog_farms(id),
    element_type TEXT CHECK (element_type IN ('artifact', 'seed', 'thinker', 'theory')),
    element_id UUID NOT NULL,
    added_at TIMESTAMPTZ DEFAULT NOW(),
    added_by TEXT CHECK (added_by IN ('user', 'ai')),
    confidence NUMERIC(3,2) -- si fue AI
);

-- Agujeros de Gusano (conexiones no obvias)
CREATE TABLE cog_wormholes (
    id UUID PRIMARY KEY,
    project_id UUID REFERENCES cog_projects(id),
    source_type TEXT,
    source_id UUID,
    target_type TEXT,
    target_id UUID,
    isomorphism_description TEXT,
    confidence NUMERIC(3,2),
    status TEXT CHECK (status IN ('suggested', 'accepted', 'rejected')),
    suggested_by TEXT, -- 'gemini', 'claude', 'user'
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Métricas de tráfico (nodos más visitados/conectados)
CREATE TABLE cog_node_metrics (
    id UUID PRIMARY KEY,
    element_type TEXT,
    element_id UUID,
    visit_count INTEGER DEFAULT 0,
    connection_count INTEGER DEFAULT 0,
    last_visited TIMESTAMPTZ,
    heat_score NUMERIC(5,2) -- calculado
);
```

## 🌊 Metáfora Visual Unificadora

El **océano de conocimiento**:

- **Superficie** = Vista constelaciones (puntos de luz)
- **Corrientes** = Conexiones entre nodos
- **Remolinos** = Granjas (atractores semánticos)
- **Profundidad** = Zoom in a nodos individuales
- **Agujeros de gusano** = Atajos entre zonas distantes
- **Surfear** 🏄 = Navegar dejándose llevar por las corrientes

---

## 💜 Filosofía de Implementación

> *"La educación no es acumulación lineal, sino conexiones inesperadas según áreas de interés."*

1. **Nunca lista plana** - Siempre con contexto relacional
2. **AI como jardinero** - Sugiere, no impone
3. **Usuario como curador** - Valida, rechaza, conecta
4. **Fractal siempre** - Zoom in/out infinito
5. **Alucidar como verbo** - Explorar lo no-obvio
